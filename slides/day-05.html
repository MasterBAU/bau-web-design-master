Sublime Text Tip

Package Control

El Package Control de Sublime Text es un gestor de paquetes que nos permite instalar muy facilmente cientos de plugins reailzados por diferentes desarrolladores.

Para instalarlo, debemos dirigirnos a la siguiente dirección:

https://sublime.wbond.net/installation

Una vez allí copiamos el código correspondiente a nuestra versión y lo ejecutamos en la consola de nuestro programa.

Sublime Text > View > Show Console

Luego de ejecutado el código, reiniciaremos Sublime Text y ya podremos acceder al Package Control.

Para instalar un nuevo plugin, lo haremos a través de la Paleta de Comandos, donde escribiremos "Install Package" y luego el nombre del plugin a instalar.

Accederemos a la paleta con:

Cmd + Shift + P

Posicionamiento mediante
float

La propiedad float modifica el flujo natural del elemento y lo "flota" situándolo a la izquierda o derecha de su contenedor permitiendo al resto de elementos inline ocupar el espacio colindante.

Inicialmente se creó para posicionar imágenes dentro de un bloque de texto pero con el tiempo acabo por ser una de las herramientas CSS más usadas para generar layouts.

La propiedad clear

Omite el float del elemento anterior.

clear soporta 4 valores:

  • none: el elemento se verá afectado normalmente por un float.
  • left: el elemento se desplazará abajo cuando se encuentre con un float: left.
  • right: el elemento se desplazará abajo cuando se encuentre con un float: right.
  • both: el elemento se desplazará abajo ante cualquier float.

Clase .Clearfix

Una buena práctica es aplicar la clase .clearfix al contenedor con elementos flotados:

.clearfix:before,
.clearfix:after {
    content: " ";
    display: table;
}
.clearfix:after {
    clear: both;
}
.clearfix {
    *zoom: 1;
}
