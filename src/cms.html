¿Qué es un CMS?

CMS son las siglas de Content Management System (sistema de gestión de contenidos). Un sistema de gestión de contenidos web (WCMS o Web Content Management System) es una aplicación web que nos permite crear y administrar los contenidos para nuestra web, normalmente ofreciendo una interfaz para esta gestión de datos y realizando de manera transparente para el usuario las gestiones necesarias con la base de datos.

Preguntas para decidir que CMS se adapta mejor a nuestras necesidades

  • ¿Necesitamos uno?
  • ¿Para qué es el proyecto?
  • ¿Cuanta personalización y dinamismo de contenidos necesita la web?
  • ¿Tenemos algún requisito técnico? ¿El cliente ya tiene un hosting, por ejemplo?
  • ¿Tenemos ya experiencia con algun CMS?
