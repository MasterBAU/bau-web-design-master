Clear: both;

Una técnica sencilla para solucionar el problema es colocar dentro del contenedor, a continuación de los elementos flotados, un elemento con la propiedad
clear: both

1
2
3
clear: both;

Overflow: auto;

Otra posible solución, es recurrir a la propiedad overflow.
Cualquier valor diferente a visible nos corregirá el problema 

div { overflow: auto; }

Clase .Clearfix

Una buena práctica es aplicar la clase .clearfix al contenedor con elementos flotados:

.clearfix:before,
.clearfix:after {
    content: " ";
    display: table;
}
.clearfix:after {
    clear: both;
}
.clearfix {
    *zoom: 1;
}
