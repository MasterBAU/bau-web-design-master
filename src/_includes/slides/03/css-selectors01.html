Tipos de selectores

Etiqueta / Elemento

Etiqueta / Elemento

Es el más genérico.
Aplicamos estilo a todos los elementos de la página.

Clase

  • Atributo HTML aplicable a cualquier elemento
  • Pueden tener cualquier nombre
  • Se pueden aplicar a múltiples elementos en una misma página
  • Son sensibles a las mayúsculas

Identificador

  • Atributo HTML aplicable a cualquier elemento
  • Pueden tener cualquier nombre
  • Los ID han de ser únicos en una misma página
  • Son sensibles a las mayúsculas
  • Recuerda: Clases > estilos | IDs > Javascript

Combinaciones de selectores

Aumentamos la especificidad

Selector Elemento-Específico

La combinación Elemento-específico aplica una clase o un ID exclusivamente al elemento que le precede.

Selector Descendente

  • Especifican un elemento basándose en su jerarquía
  • Ofrece altos niveles de especificidad pero es poco versátil
  • Se colocan las etiquetas una a continuación de la otra separadas por un espacio

Agrupar selectores

Aplicamos los mismos estilos situándolos uno a continuación del otro
separados por coma (,)

Selector contextual

Selector universal

Aplica la declaración a todos los elementos de la página sin excepción

Selector atributo

El selector de atributo aplicará la declaración sólo cuando el elemento disponga de un atributo que coincida exactamente con el contenido entre paréntesis [ ]

Selector atributo + valor

El selector de atributo aplicará la declaración sólo cuando el elemento disponga de un atributo y un valor que coincida exactamente con el contenido entre paréntesis [ ]

Selector de hijos

Selecciona a un elemento que se encuentra anidado.

Selector de hermanos adyacentes

Selecciona al elemento indicado que se encuentre a su mismo nivel en la jerarquía y a continuación.

Selector de hermanos NO adyacentes

Selecciona al elemento indicado que se encuentre a su mismo nivel independientemente de la jerarquía.
