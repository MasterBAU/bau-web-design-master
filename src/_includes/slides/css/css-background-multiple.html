Multiples imágenes de fondo

Podemos utilizar más de una imágen de fondo simplemente separando los valores con una coma.

Si asignamos más de una propiedad relativa al backgrouns tenemos que cuidar que el orden sea el mismo para todas las propiedades.

Si algunas de las imágenes se superponen será la primera de la lista la que quede al frente y el resto le seguirán en orden.


div {
    background:
        url(day-06/dalek.png) 140px top,
        url(day-06/megaman.png) 90px 100px;

        background-repeat: no-repeat;
}
