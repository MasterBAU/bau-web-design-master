Elementos estructurales

Etiqueta Elemento que representa
<div> y <span> Elementos sin contenido semántico. El primero es inline, mientras que el segundo es block.
<header> Cabecera de un documento o de una sección del documento. Suele contener un elemento <h1...h6>
<footer> Pie de un documento o de una sección del documento.
<nav> Navegación de un documento o página. Contiene enlaces a otros documentos o a secciones del mismo documento.
<article> Sección de un documento cuyo contenido tenga sentido por si mismo independientemente de su contexto.
<section> Un grupo de contenido de temática relacionada.
<aside> Sección de una página con contenido relacionado o complementario al contenido principal.
