Sublime Text Tip

Installing themes

Abrimos la consola mediante: Cmd + Shift + p
Escribimos: Install package + Enter
En la ventana que nos aparece escribimos: Theme - [nombre]
Seleccionamos un tema: Prefs - Color Scheme - Theme [nombre]
