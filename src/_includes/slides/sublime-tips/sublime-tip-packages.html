Sublime Text Tip

Package Control

El Package Control de Sublime Text es un gestor de paquetes que nos permite instalar muy facilmente cientos de plugins reailzados por diferentes desarrolladores.

Para instalarlo, debemos dirigirnos a la siguiente dirección:

https://sublime.wbond.net/installation

Una vez allí copiamos el código correspondiente a nuestra versión y lo ejecutamos en la consola de nuestro programa.

Sublime Text > View > Show Console

Luego de ejecutado el código, reiniciaremos Sublime Text y ya podremos acceder al Package Control.

Para instalar un nuevo plugin, lo haremos a través de la Paleta de Comandos, donde escribiremos "Install Package" y luego el nombre del plugin a instalar.

Accederemos a la paleta con:

Cmd + Shift + P
